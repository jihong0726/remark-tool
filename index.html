专属客服备注生成助手（AI版）

输入用户第一句话 + 选择处理动作，用 AI 帮你生成标准四段备注。
0️⃣ OpenAI API Key 只在本地浏览器使用，不会保存、不会上传到 GitHub。
注意：请不要把 Key 写进代码或截图分享。每位使用者用自己的 Key 即可。
1️⃣ 填写基础信息 建议先贴用户第一句话 / 关键信息
暂未识别，如有需要可手动选择。
解释类 & 用户已理解 → 多数为「否」，升级 / 等结果 / 待补资料 → 多数为「是」。
如复制了一整段对话，也可以贴这里，AI 会自动提炼成简短「问题描述」。
2️⃣ 选择本次处理动作（可多选） 将被组合成简短「处理方案」一句话
例如同时勾选「向用户索取资料 + 正常安抚 + 升级处理」，AI 会用内部话术帮你总结。
3️⃣ 生成备注 推荐先用 AI 生成，再人工微调
